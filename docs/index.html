Quick Start

Up and Running in Minutes

Get the platform running locally with just a few commands. Docker handles all the complexity.

1

Clone Repository

git clone https://github.com/Div-9898/AIO_Final_Project.git
2

Configure Environment

cp .env.example .env && nano .env
3

Start Platform

./start.sh start

GET /api/v1/vehicles

List all vehicles with current status and position

GET /api/v1/vehicles/{id}

Get specific vehicle details including history

GET /api/v1/vehicles/{id}/history

Get vehicle location history with timestamps

POST /api/v1/optimization/start

Start a new route optimization job

GET /api/v1/optimization/{run_id}

Get optimization status and results

POST /api/v1/optimization/cancel/{run_id}

Cancel a running optimization job

GET /api/v1/risk/scores

Get current risk scores for all vehicles

GET /api/v1/risk/alerts

Get active risk alerts

POST /api/v1/risk/acknowledge/{alert_id}
