Client Layer

Next.js 14 with App Router for server-side rendering and optimal performance

Next.js 14

React Framework

Server-side rendering, App Router, and React Server Components for optimal performance and SEO.

Mapbox GL + Deck.gl

Mapping & Visualization

WebGL-powered interactive maps with 3D terrain, buildings, and high-performance data visualization for 1000+ vehicles.

Zustand

State Management

Lightweight, fast state management with minimal boilerplate. Perfect for real-time data synchronization.

Socket.IO Client

Real-time Communication

Bidirectional event-based communication for live vehicle tracking, alerts, and messaging.

API Gateway

FastAPI with async support for high-throughput request handling

FastAPI

Web Framework

Modern, fast Python web framework with automatic OpenAPI documentation and async/await support.

Google OR-Tools

Optimization Engine

Industry-standard Vehicle Routing Problem solver with constraint programming and multiple optimization strategies.

PyTorch + SHAP

ML & Explainability

Machine learning models for risk prediction with SHAP-based explainability for transparent decision-making.

Fairlearn

Fairness & Bias

Algorithmic fairness assessment with demographic parity metrics and automated bias detection.

Data Layer

Polyglot persistence with specialized databases for each use case

PostgreSQL 16

Primary Database

ACID-compliant relational database for operational data, vehicles, routes, and historical records.

Redis 7

Cache & Pub/Sub

In-memory data store for caching, session management, and real-time message brokering via pub/sub.

Neo4j 5.18

Graph Database

Native graph database for stakeholder relationships, influence mapping, and network analysis.

Google Gemini

AI Services

Large language model integration for ethical scenario generation, policy creation, and natural language processing.

Data Flow

How Data Moves

Understanding the real-time data pipelines

1

Real-Time Vehicle Updates

Vehicle simulation generates position data → Redis Pub/Sub → Socket.IO Server → Frontend Map

2

Route Optimization

User configures constraints → FastAPI → OR-Tools Solver → Streaming progress via WebSocket → Final optimized routes

3

Risk Assessment

Sensor data (weather, traffic, vehicle health) → PyTorch ML Models → Risk scores → Alerts via WebSocket

4

Sentiment Analysis

Customer message → NLP Processing → Sentiment classification → Dashboard metrics update

5

Stakeholder Analysis

Relationship data → Neo4j Graph → Cypher queries → Force-directed visualization in frontend
