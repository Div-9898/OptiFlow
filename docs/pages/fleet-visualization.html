Core Feature

Real-Time Fleet Visualization

Track every vehicle in your fleet with sub-second precision using interactive 3D maps, live position updates, and high-performance rendering.

Interactive Mapbox GL Maps

Experience your fleet like never before with our Mapbox GL integration. Navigate through your operational area with smooth pan, zoom, and rotation controls, all rendered in beautiful 3D with terrain visualization.

  • 3D Terrain Rendering - Visualize elevation and topography for better route understanding
  • Custom Map Styles - Switch between light, dark, satellite, and custom themes
  • Building Footprints - 3D buildings in urban areas for precise delivery visualization
  • Traffic Layer - Real-time traffic conditions overlaid on routes

Live Vehicle Tracking

Every vehicle in your fleet sends position updates via WebSocket, ensuring you always see the most current location. Animated transitions make movement smooth and easy to follow.

  • Sub-Second Updates - Position data refreshed every 1000ms via WebSocket
  • Animated Transitions - Smooth movement between position updates
  • Status Indicators - Color-coded markers showing vehicle status (active, idle, issue)
  • Click-to-Details - Click any vehicle for detailed information panel

High-Performance Rendering

Using Deck.gl's WebGL-powered rendering engine, we can display thousands of vehicles simultaneously without compromising performance or user experience.

  • 1000+ Vehicles - Render massive fleets at 60 FPS
  • GPU Acceleration - WebGL-based rendering for maximum performance
  • Level of Detail - Automatic detail reduction when zoomed out
  • Clustering - Intelligent grouping of nearby vehicles at low zoom levels

Route Visualization

See the planned routes for each vehicle with color-coded segments indicating completion status, delays, and estimated time of arrival.

  • Animated Routes - Watch vehicles progress along their routes in real-time
  • Color Coding - Green (on-time), yellow (slight delay), red (significant delay)
  • Stop Markers - Clear indication of delivery/pickup locations
  • ETA Display - Estimated arrival time at each stop
Implementation

WebSocket Integration

Simple code to connect and receive real-time vehicle updates

fleet-tracking.tsx 
// Connect to WebSocket for real-time vehicle updates
import { useEffect, useState } from 'react';
import { io } from 'socket.io-client';

export function useVehicleTracking() {
  const [vehicles, setVehicles] = useState<Vehicle[]>([]);

  useEffect(() => {
    const socket = io('http://localhost:8000');

    socket.on('vehicle:update', (data: VehicleUpdate) => {
      setVehicles(prev =>
        prev.map(v =>
          v.id === data.id
            ? { ...v, lat: data.lat, lng: data.lng, status: data.status }
            : v
        )
      );
    });

    socket.on('vehicle:status', (data: StatusUpdate) => {
      // Handle status changes (active, idle, issue)
      console.log(`Vehicle ${data.id}: ${data.status}`);
    });

    return () => socket.disconnect();
  }, []);

  return vehicles;
}
